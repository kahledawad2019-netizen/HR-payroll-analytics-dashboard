# HR & Payroll Analytics Dashboard

## 📊 Project Overview
This project focuses on workforce analytics, using employee-level data to explore salary structures, tax deductions, bonuses, and net salary distribution.  
It provides actionable insights into payroll management and workforce composition.

## 🔑 Key Features
- **Employee Data Analysis:**
  - Salary, tax, bonus, and net salary calculations
  - Extra bonus eligibility based on net salary thresholds
- **Workforce Segmentation:**
  - Departmental breakdown
  - Work types (Full-time, Contract, Hourly, Half-time)
  - Country-level workforce distribution
- **Hiring Trends:**
  - Hiring year, month, and experience analysis

## 🛠️ Tools & Skills Demonstrated
- Excel data cleaning and transformation
- HR analytics and payroll calculations
- Segmentation and workforce distribution analysis
- Reproducible workflow design

## 🚀 Insights
- Highlighted salary distribution across departments and countries.
- Identified employees eligible for extra bonuses.
- Showed hiring trends and workforce composition by work type.
